# CS50

Solutions to selected programming projects from **Harvard's CS50x: Introduction to Computer Science**.

This repository contains implementations of algorithms, data structures, systems programming, image processing, and full-stack web development projects completed throughout the course.

## Projects

| Project | Concepts |
|---------|----------|
| Finance | Flask, SQLite, Authentication, REST APIs, Web Development |
| Tideman | Graph Algorithms, Cycle Detection, Ranked-Choice Voting |
| Speller | Hash Tables, Linked Lists, Dynamic Memory, File I/O |
| Filter | Image Processing, Convolution, BMP Manipulation, C Programming |

## Technologies

- C
- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja
- SQL

## Notes

These projects are selected from CS50x and demonstrate core computer science concepts including algorithms, data structures, memory management, web development, and systems programming.
